# React 核心知识点示例

## 主要功能
- 函数组件使用示例
- `useState` Hook
- 事件处理 (`onClick`)
- JSX 语法

## 测试说明
测试用例覆盖以下功能：
1. 初始消息渲染
2. 消息反转功能
3. 计数增加功能

## 运行测试
```bash
npm install
npm test
```

1. **函数组件**: React的现代组件编写方式
2. **Hooks**: `useState`管理组件状态
3. **事件处理**: 通过props传递事件处理器
4. **JSX**: JavaScript语法扩展，描述UI结构
5. **测试**: 使用React Testing Library测试组件行为
