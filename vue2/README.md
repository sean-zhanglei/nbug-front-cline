# Vue 2 核心知识点示例

## 主要功能
- Options API 使用示例
- 数据绑定 (`data`)
- 方法定义 (`methods`)
- 事件处理 (`@click`)
- 样式作用域 (`scoped`)

## 测试说明
测试用例覆盖以下功能：
1. 初始消息渲染
2. 消息反转功能
3. 计数增加功能

## 运行测试
```bash
npm install
npm test
```

## 核心知识点总结
1. **Options API**: Vue 2 的主要编程方式
2. **响应式数据**: data 返回的对象会被 Vue 转为响应式
3. **方法绑定**: methods 中定义的方法可以绑定到模板事件
4. **组件作用域**: scoped 样式仅作用于当前组件
